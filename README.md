## Installation

Clone the project

```bash
  git clone https://github.com/Bala-Subrahmanyam-Reddy/petC-frontend
```

Go to the project directory

```bash
  cd petC-frontend
```

Create .env.development
```bash
REACT_APP_API_URL=<URL>
REACT_APP_AUTH_TOKEN=<Bearer token>
```

Install dependencies
```bash
npm i
```

Start the app
```bash
npm start
```
